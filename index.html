启动tomcat容器以及上传war包

![启动容器](docker4.jpg)
需要将tomcat webapps目录挂载到本地,将容器的8080端口映射到本机的8081端口,容器的webapps挂载到本地的/usr/local/tomcat/webapps目录
docker run -d --name mytomcat -p 8081:8080 -v /usr/local/tomcat/webapps:/usr/local/tomcat/webapps tomcat
不需要挂载则不需要指定 -v
docker run -d --name mytomcat -p 8081:8080  tomcat
此方式需要将war放入容器需要进入容器内部
docker exec -it 容器的id /bin/bash 
然后将war包拷贝到容器中
docker cp /java.war tomcat:/usr/local/tomcat/webapps
如需设置开机自动启动本容器
docker run -d --name mytomcat -p 8081:8080 --restart=always  tomcat
